## 使用方法
- Star并Fork此项目。
- 进入你的仓库，Settings → Secrets,
- 添加1个 Secret（利用secret解决隐私问题）：

- Name          |         Value
- USERS	        |         {"名字":["logintoken","学号"],"名字2":["logintoken2","学号2"]}

备注：
- ①理论上Value格式填写正确，可添加无限用户
- ②如何获取[logintoken](https://www.baidu.com/) 
- ③上一步的教程后续补上
- ④若完善好Server酱推送，上述Value格式将会改变
